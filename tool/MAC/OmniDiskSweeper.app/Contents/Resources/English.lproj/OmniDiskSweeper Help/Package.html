Package

A package is a collection of associated files that are installed at the same time and work together to provide functionality to the system, such as a printer driver, an update to your system software, or an application. Just like when you buy a piece of software in the store and it comes in a package, when you download software it comes all packaged up and ready to install. It is important to know whether a file is part of a package because deleting it might stop the whole application from working. Generally, when you are sweeping a hard drive, you want to focus on files that are not part of packages.
