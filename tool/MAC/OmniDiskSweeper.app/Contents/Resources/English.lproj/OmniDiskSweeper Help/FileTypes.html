File Types

As you browse your files in OmniDiskSweeper you will only see the titles of each file or folder. You can see the kind of file simply by selecting a file or folder: an icon is displayed in the lower right corner of the browser window that indicates the file type. You can also double-click this icon to show the file in the Finder or drag the icon to another application to manipulate it outside of OmniDiskSweeper.
