Selecting a Drive to Sweep

When you first start OmniDiskSweeper you need to select a hard drive that you want to clean up or "sweep." All of the hard drives connected to your computer as well as any hard drives you have access to over the network are displayed in the drive list. The drive list also shows you how much space is free on each drive. After selecting a drive, you can always return to the drive list from the Tools menu.
To select a local drive to sweep
Do one of the following:
Double-click the local drive.
Select the local drive and click the Sweep button at the bottom of the drive list.
To select a network drive to sweep
Click the disclosure triangle next to Network.
Available drives on your network are displayed. Do one of the following to select one:
Double-click the network drive.
Select the network drive and click the Sweep button at the bottom of the drive list.
You can also open a specific folder that you want to sweep by selecting File ‣ Size Folder.... A file browser dialog appears where you can locate and select the specific folder you want to sweep. Because it is very common to want to sweep your home folder, there is a menu item just for it: select File ‣ Size Home Folder.
